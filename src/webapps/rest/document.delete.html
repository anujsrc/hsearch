document.delete
REST API . document.delete

Deletes a document from the index. It also cleans up dictionary.

Parameters


Name Type Description
key String The document Id. This is the same value you have set at HDocument.key during indexing.
hkey String The API Key. If you don't have an account key, You can Create A Key Here.

Response


On sucess it returns

OK

In case of error, the error message is inside XML. An error message format is,
The Error response format is, 

        <?xml version="1.0" encoding="UTF-8"?>
        <result>
            <error>
                <msg>Failure : SearchService: ...
            </error>
        </result>
