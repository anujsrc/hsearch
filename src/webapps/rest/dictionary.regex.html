dictionary.regex
REST API . dictionary.regex

Finds words inside the dictionary based on a regular expression.

Parameters


Name Type Description
word String Regular expression. Example,

Starting with ^word
Ends with word $
hkey String The API Key. If you don't have an account key, You can Create A Key Here.

Response


On sucess it returns word listing. The XML format is as below

<result>
    <string>without</string>
</result>

In case of error, the error message is inside XML. An error message format is,
The Error response format is, 

        <?xml version="1.0" encoding="UTF-8"?>
        <result>
            <error>
                <msg>Failure : SearchService: ...
            </error>
        </result>
