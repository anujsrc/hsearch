dictionary.refresh
REST API . dictionary.refresh

Refreshes the fuzzy search repository. A regular lookup of word may show the word where as at interval of 30minutes the fuzzy repository refreshes the word list for fuzzy search. A manual refresh also will make sure latest dictionary shows up in the fuzzy search.

Parameters


Name Type Description
hkey String The API Key. If you don't have an account key, You can Create A Key Here.

Response


On sucess it returns word refreshing. The XML format is as below

OK
