HSearch REST API


The REST API enables you to interact with hadoopsearch web site programmatically via HTTP requests.

To make requests to the REST API with your API KEY, you requests URLs in the following format:


Either using a GET method 
../service.html?
        hkey=API KEY&
        service= [ search | management ]&
        action=  [ method ]&
        arg0=value0&
        arg1=value1
Or a more secure POST method
URL: ../service.html
POST Parameters
service=[ search | management ]
action=[ method ]
hkey=API KEY
arg0=value0 arg1=value1

For example, you can search documents with following REST call:

../service.html?hkey=50BFB2F8337ADC32FC5866EFF998D608&service=search&action=document.search&query=hello

Administrative Methods (Service = management)

account.get

Gets detail information about the account.

account.buckets.set

Allocate space(buckets) for indexing where each bucket keeps 32178 documents or records.

account.bucket.truncate

Deletes all documents and records inside a bucket.

account.truncate

Deletes all records and documents of an account.

Application Setup Methods (Service = management)

account.doccodes.get

Shows all document types for the index.

account.doccodes.set

Allows to map simillar document types.

account.doccodes.append

Append a new document type to the existing document type list.

account.termcodes.get

Shows all document fields.

account.termcodes.set

Allows to map simillar document fields.

account.termcodes.append

Append a new field.

account.fieldweight.get

Shows all allocated field weights.

account.fieldweight.set

Setup the field weights.

account.fieldweight.append

Append a new field weight.


Indexing Methods (Service = search)

document.add

Add a document to the index in real time.

document.get

Refer to document detail using document id.

document.addXml

Add a XML document to the index where xml elements are extracted and treated as field.

document.load

Load a CSV or TSV file to the index in real time.

document.batch

Add list of documents in one go.

document.delete

Delete a document given a document id.

document.deletes

Delete documents in batch for a set of documents.


Searching Methods (Service = search)

document.search

Search for documents/records. Read details to know more about hsearch search syntax.


Dictionary Methods (Service = search)

dictionary.add

Add a word to dictionary with synonums and other details.

dictionary.lookup

Lookup for a specific word detail in dictionary.

dictionary.refresh

Refresh the dictionary to enable fuzzy search.

dictionary.list

Lists all words maintained in dictionary.

dictionary.addAll

Add multiple entries to the dictionary.

dictionary.spell

Do a fuzzy search of a word.

dictionary.regex

Perform a regular expression search in the dictionary.

dictionary.delete

Delete a entry from dictionary.
